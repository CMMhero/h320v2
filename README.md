# Hero Two

**A bot created by CMMhero**

discord.js, commando

Initial Setup
--
1. Download the zip or use gitclone
	```
	git clone https://github.com/CMMhero/h320v2
	```

2. Install all the modules
	```
	npm i
	```

3. Create a file named "config.json"
	```
	{
 	 "token": "bot token",
 	 "prefix": "command prefix",
 	 "mongoPath": "mongo path",
	}
	```

4. Run the bot
	```
	node index.js
	```
